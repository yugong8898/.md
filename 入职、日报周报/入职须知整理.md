# 入职须知整理

## 一、工作时间

- 工作时间：9:00 - 18:00
- 弹性时间：0.5 小时（动态）
- 午休时间：12:00 - 13:00

## 二、社保

- 每月 15 日前入职，当月缴纳社保

## 三、加班规则

### 计算单位

- 起算单位：工作日 1.5 小时 / 休息日 9 小时
- 累算单位：0.5 小时

### 工作日加班

| 条件 | 说明 |
|------|------|
| 起算时间 | 20:00 开始计算加班（即加班满 1.5 小时起算） |
| 餐补+交通补助 | 超过 21:00 且次日 12:00 前到岗：餐补 30 元 + 交通补助 30 元 |
| 超过 23:00 | 交通补贴全额报销 |
| 累算单位 | 0.5 小时 |

### 休息日加班

| 条件 | 说明 |
|------|------|
| 补偿方式 | 只能调休（不可折现） |
| 起算时长 | 9 小时 |
| 餐补+交通补助 | 18:00 以后：餐补 30 元 + 交通补助 30 元 |
| 超过 23:00 | 交通补贴全额报销 |
| 累算单位 | 0.5 小时 |

### 加班时长参考

| 级别 | 时长范围 |
|------|----------|
| 一般 | ~30 小时 |
| 较多 | ~66 小时 |
| 高强度 | 66 小时+ |
