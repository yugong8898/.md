## 二、改动清单

### 区域一：网络货运主体栏

**当前布局**：
```
[主体下拉，含「全部」] [查看充值账号按钮] [我的银行卡按钮] [（未绑卡文字提示）]
```

**改后布局**：
```
[主体下拉，无「全部」] [我的银行卡/绑定银行卡 链接] [设置支付密码]
```

改动：

1. 删掉 `<el-option label="全部" value="" />`

2. 进入页面默认选中最近一笔运单的网络货运主体（依赖待确认事项 #1）：
   - 后端接口返回标记：取 `options` 中 `isDefault === true` 的那条

4. 「我的银行卡」改为链接，未绑卡时文案改「绑定银行卡」+ 红色，删掉原有提示文字：
```html
<!-- 原代码 -->
<el-button type="primary" @click="handleBindCard">我的银行卡</el-button>
<p v-if="!isBindCard" class="bind-card-tip">（未绑卡，请您提前绑定银行卡。）</p>

<!-- 改后 -->
<el-button type="text" :style="!isBindCard ? 'color: red;' : ''" @click="handleBindCard">
  {{ isBindCard ? '我的银行卡' : '绑定银行卡' }}
</el-button>
```

5. `<pay-pwd-dialog />` 从现金账户信息 title 区域移到银行卡链接后面

### 区域三：充值收款账户（右侧卡片）

**当前状况（有问题）**：右侧卡片 title 写「充值收款账户」，但绑定的是 `infoCredit` 授信数据，内容与 title 不符。真实充值账户数据（`aaccountName` / `asubBranch` / `aaccountNumber` / `sbn`）在 `accountInfoDialog` 弹窗里，来自 `options` 主体列表。

**改后布局**：
```
充值收款账户
账户名称：xxx
收款账号：xxx
开户行：xxx
联行号：xxx
[复制收款账户信息]  [复制账户名称和收款账号]
```

改动：

1. 右侧卡片替换为真实充值收款账户字段，数据取当前选中主体在 `options` 中的记录，随主体切换联动

2. `infoCredit` 授信数据不在此展示，迁至轮播图下方「授信账户信息」模块（非微前端，见区域五）

3. 新增「复制收款账户信息」按钮，复制账户名称 + 收款账号 + 开户行 + 联行号：
```javascript
copyAllInfo() {
  const { aaccountName, aaccountNumber, asubBranch, sbn } = this.currentSubject
  const text = `账户名称：${aaccountName}\n收款账号：${aaccountNumber}\n开户行：${asubBranch}\n联行号：${sbn}`
  navigator.clipboard.writeText(text).then(() => {
    this.$message({ message: '复制成功', type: 'success', duration: 3000, showClose: true })
  })
}
```

4. 新增「复制账户名称和收款账号」按钮，仅复制账户名称 + 收款账号：
```javascript
copyNameAndNo() {
  const { aaccountName, aaccountNumber } = this.currentSubject
  const text = `账户名称：${aaccountName}\n收款账号：${aaccountNumber}`
  navigator.clipboard.writeText(text).then(() => {
    this.$message({ message: '复制成功', type: 'success', duration: 3000, showClose: true })
  })
}
```

`currentSubject` 为新增计算属性，根据 `subjectCode` 从 `options` 中取当前主体：
```javascript
computed: {
  currentSubject() {
    return this.options.find(o => o.companyCode === this.subjectCode) || {}
  }
}
```
